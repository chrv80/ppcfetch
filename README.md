# ppcfetch
ppcfetch is a simple console tool for Mac OS X PPC that provides basic system information.
Written in C. Built and tested on Leopard. 

Made just for fun.

To install: clone this repository, run "make", then "make install". The default installation location is /usr/sbin, it can be changed in makefile. 

To uninstall: run "make uninstall".
